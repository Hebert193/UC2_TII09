No HTML, os elementos podem ser exibidos de duas maneiras principais: inline e block. essas duas propriedades definem como os elementos são renderizados no fluxo da pagina

Elementos block

  • Definição: Um elemento block ocupa toda a largura disponivel de seu conteiner pai e começa em uma nova linha
  • Exemplos: div, h1, p, section, article.

Elementos Inline

  • Definição:Um elemento inline ocupa apenas o espaço necessario para o seu conteudo e não começa em uma nova linha
  • Exemplos:span , a, strong, em, img.

Mudando o comportamento do displey com CSS

É possível modificar o comportamento dos elementos, usando a propriedade display no CSS: 

            
    div {
        display: inline;
    }

    div {
        display: block;
    }
