aula tabelas

introduçao as tabelas em html

tabelas sao estruturas em html que permitem a organizacao de dados em linhas e colunas. sao comumente usadas para exibir calendarios ,horarios, lista de preco ,etc

estrutura basica de uma tabela

a estrutura basica de uma tabela em html e compodta por varias tags:

  • table:define o inicio e fim de uma tabela.
  • tr:define uma linha na tabela.
  • td:define uma celula dentro de uma linha (daods)
  • th:define uma celula de cabecalho

exemplo 1: tabela basica

nome idade profissao
ana 28 engenheira
bruno 34 medico

explicação

  • nesse exeplo, usamos th para o cabeçalho e td para os dados.
  • o atributo border="1" e usado para adicionar bordas a tabela
diciplina un1 un2 un3 un4
portugues 7 8 5 5
matematica 8 5 5 7
ingles 7 8 8 7

mesclagem de celulas

a mesclagem de celulas pode ser feita tanto em colunas quanto em linhas

  • colspan: mescla celulas horizontalmente (colunas)
  • rowspan: mescla celulas verticalmente (linhas)
nome idade profissao
carlos 30 professor
30 arquiteto
daniela designer grafico

exeplicação:

  • rowspan="2": a celula "carlos" se expande verticalmente para cobrir duas linhas
  • colspan="2":a celula "designer grafico" se expande horizontalmente para cobrir duas colunas.

adicionando um cabeçalho e rodape a tabela

alem das linhas de dados , uma tabela pode ter um cabeçalho ( thead ), um corpo ( tbody ) e um rodape ( tfoot )

produto quantidade preço unitario
caneta 10 R$2,00
caderno 5 R$15,00
total R$ 95,00

exeplicação:

  • thead: define o cabeçalho de tabela
  • tbody:define o corpo da tabela , onde ficam os dados
  • tfoot:define o rodape ,ideal para somarios ou resumos

tabela aninhada

e possivel colocar uma tabela dentro de outra celula de tabela. isso e conhecido como aninhamento de tabela.

exemplo de tabela aninhada:

produto detalhes
laptop
modelo preço
apple m1 R$7.000,00
