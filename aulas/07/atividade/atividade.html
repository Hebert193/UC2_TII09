sintese das propriedades css

1.color

define a cor de um elemento,

 h1 { 
     color: blue ; 
 }

2. text-align

alinha o texto dentro de um elemento.

  • left: alinha o texto a esquerda (padrao)
  • center: centraliza o texto
  • ringht: alinha o texto a direita.
 h1 {
     text-aling: center;
 }

3.fonte-size

define o tamanho da fonte de um elemento.

 p {
      fonte-size: 18px;
 }

4.background-color

define a cor de fundo de um elemento

 button {
     background-color : yellow;
}

5. color (para botoes)

define a cor do texto de um elemento (usado em conjunto com background-color.)

button {
     color: black;
 }
