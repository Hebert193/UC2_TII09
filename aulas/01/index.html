HTML

HTML (linguagem de marcaçao de hipertexto) é o bloco de construção mais básico da web. define o significado e a assinatura de conteudo web.

TAG ou Marcação

O HTML usa "marcação" para anotar texto, imagem e outros conteudos para exibição em um navegador web

um elemento HTML é separado de outro texto em um documento por "tags". As tags consistem no nome do elemento entre um "<" e ">" .
